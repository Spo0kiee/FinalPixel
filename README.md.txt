# 🛒 FinalPixel – BfK Projekt Webshop

Willkommen zum offiziellen Repository für mein Webshop-Projekt „FinalPixel“, erstellt im Rahmen des BfK-Projekts 2025.

## 🚀 Features
- 🏰 Mittelalterliches Theme
- 🧩 Produktsystem mit Genre/Plattform-Filterung
- 🔍 Dynamische Detailseiten (produkt.php)
- 🧭 Navigierbares Frontend mit Header, Footer & Plattformwahl
- 💾 MySQL-Anbindung
